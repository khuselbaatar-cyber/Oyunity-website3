# Oyunity Website

Premium Next.js 14 (App Router) website for Oyunity — Elite Scholarship Mentorship Ecosystem.

## 🚀 Localhost дээр ажиллуулах

### 1. Dependencies install

```bash
npm install
```

(Эсвэл `yarn install` / `pnpm install`)

### 2. Development server эхлүүлэх

```bash
npm run dev
```

Дараа нь browser-аас [http://localhost:3000](http://localhost:3000) нээх.

### 3. Production build

```bash
npm run build
npm start
```

## 🎨 Brand

- **Primary blue:** `#1E3A8A` (`oyunity-blue`)
- **Accent gold:** `#C9A961` (`oyunity-gold`)
- **Dark navy:** `#0B1B3B` (`oyunity-navy`)
- **Soft background:** `#FAFAF8` (`oyunity-soft`)
- **Fonts:** DM Serif Display (headings) + Inter (body), Cyrillic subset-тэй

## 📦 Tech stack

- Next.js 14 (App Router)
- React 18
- Tailwind CSS 3.4
- Relume UI components
- Framer Motion
- React Icons
